# Todo Application

## Features

- View all todos
- Add new todo
- Delete todo
- View individual todo details
- REST API using Express
- Data stored in JSON file

## Technologies Used

- React
- Node.js
- Express.js

## Project Structure

todo-app/
├── frontend/
├── backend/
├── README.md
└── API.md

## Running the Backend

```bash
cd backend
npm install
node server.js
```

Runs on:

http://localhost:5000

## Running the Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on:

http://localhost:5173
